Live currency rates conversion for Real Estate Manager

== Description ==

This addon let you convert the default currency to any other currency fetching the live exchange rates. You can customize the currencies to show and a widget is also available to display the currency switcher menu. The currencies dropdown can also be added directly to the theme’s menu.
